# bilibili-live-php
a php script for live.bilibili.com

## Features
- sign
- get free silver
- get daily gift
- send daily gift to performer (roomid needed)

## Dependences
- php-curl  
- php-GD

## Usage
php bilibili-live.php  
php online.php  
#### note: online.php should be run every 5 miniutes, so 'crontable' is recommended
